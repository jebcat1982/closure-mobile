closure-mobile
--------------

A JavaScript touch/mobile library for Google's Closure framework.

Based on Model-View-Presenter(MVP) pattern.

### Demo (WebKit only):
http://nextinterfaces.com/CLOSURE

Running the demo:
-----------------
* `./plovr.sh`
* Or `java -jar ./lib/plovr.jar serve plovr.js`
